## 📚 Learning More About Palindromic Squares

peakguard explores complex math concepts. Here are some terms related to the app:

- **Palindrome**: A number or sequence that reads the same forwards and backwards.
- **Palindromic square**: A number whose square is a palindrome.
- **Carry propagation**: The process when values exceed a digit base limit and "carry" over to the next digit.
- **Numeral base**: The number of unique digits used to represent numbers (e.g., base 10 is decimal).
- **Bigint**: A way to do math with very large numbers.
- **Repunits**: Numbers consisting of only the digit 1, like 11 or 111.
- **Convolution**: A math operation used to combine sequences, relevant for carry effects.

Understanding these ideas helps you see the special cases peakguard highlights.
